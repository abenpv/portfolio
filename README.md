# Portfolio Website

Dit project is mijn persoonlijke portfolio, gemaakt als onderdeel van een schoolopdracht. De website is ontworpen om mijn vaardigheden, projecten en groei als developer te laten zien.

## 🚀 Over het Project

Het doel van deze opdracht was het creëren van een responsive website die mijn werk op een professionele manier presenteert. De focus ligt op een schoon design, overzichtelijke code en een goede gebruikerservaring op alle apparaten.

### Gebruikte Technieken
* **HTML5:** Voor de semantische structuur van de pagina's.
* **CSS3:** Voor de volledige styling en layout (Flexbox/Grid).
* **JavaScript:** Voor interactieve elementen en functionaliteit.
* **Responsive Design:** De website schaalt mee van desktop naar mobiel.

## 📂 Project Structuur

De repository is als volgt georganiseerd:

* `assets/images/`: Bevat alle gebruikte afbeeldingen, logo's en iconen.
* `css/`: Bevat de stylesheet(s) voor de vormgeving.
* `js/`: Bevat de JavaScript bestanden voor interactiviteit.
* `portfolio/`: Bevat specifieke bestanden of sub-pagina's van projecten.
* `README.md`: Deze documentatie.

## 🛠️ Installatie & Gebruik

Volg deze stappen om het project lokaal te bekijken:

1.  **Clone de repository:**
    ```bash
    git clone [https://github.com/abenpv/portfolio.git](https://github.com/abenpv/portfolio.git)
    ```
2.  **Navigeer naar de map:**
    ```bash
    cd portfolio
    ```
3.  **Open de website:**
    Open het bestand `index.html` in je browser naar keuze.
